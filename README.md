# Web Development Practice

    This repository contains my learning and practice work while studying web development.

## Technologies & Concepts Covered

    - HTML
      - Structure, semantic elements

    - CSS
      - Styling, layouts
      - Flexbox, Grid
      - Responsive design (media queries)
      - Hover effects, transitions, basic animations

    - JavaScript
      - Basics
      - DOM Manipulation
      - Event handling
      - Closures, Promises
      - Async/Await
      - Basic API calls
